# Spotify Clone

This is a **frontend clone** of the Spotify Web Player using only **HTML and CSS**. 
It looks quite similar to the actual Spotify desktop version.



# Features

- Left sidebar with Home, Search, and Your Library
- Music cards like Recently Played, Trending, and Featured Charts
- Fixed bottom music player design
- Fully responsive (hides some elements on small screens)
- Font Awesome icons and Google Fonts used



# Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts (Montserrat)


# Files and Folders

- `index.html` → The main structure of the app
- `style.css` → All the styling of the layout
- Image files → Icons and song thumbnails
- No JavaScript used in this version


# Note

🚫 This is only the **UI clone**. No music will play. It's made just to practice HTML and CSS layout skills.
